# Smart Inventory Optimization System

## Overview
This project predicts product demand, detects inventory risks, and recommends reorder quantities for supermarket chains.

## Features
- Demand Forecasting
- Stockout Detection
- Overstock Detection
- Wastage Risk Detection
- Reorder Recommendations
- Interactive Dashboard

## Tech Stack
Python, Streamlit, Machine Learning, Pandas, Plotly

## Run Locally
pip install -r requirements.txt
streamlit run app.py
